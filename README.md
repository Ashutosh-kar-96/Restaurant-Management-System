# 🍽️ Restaurant Management SaaS (MERN Stack)

## 🚀 Overview
This is a full-stack Restaurant Management System built using MERN stack.  
It supports multiple branches, real-time operations, and payment handling.

---

## 🔥 Features
- Multi-tenant system (Admin + Branch users)
- Role-Based Access Control (JWT authentication)
- Order & billing management system
- Real-time updates using Socket.IO
- Payment integration (Stripe)
- REST APIs with scalable backend

---

## 🧠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MySQL + Prisma ORM
- Real-time: Socket.IO
- Queue: BullMQ / RabbitMQ
